# 🚀 EZLearn — Master Web Development

> Beautiful, bite-sized lessons for modern web development.

## 📖 Overview

EZLearn is a lightweight, modern web development learning platform built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just clean code.

---

## ✨ Features

### 📚 8 Comprehensive Courses
- React, Angular, Laravel, Node.js, Vue.js, Python, CSS, TypeScript
- Each course has 5-8 topics with multiple lessons
- Real code examples with plain-English explanations

### 🎨 Premium Design
- Glass morphism UI with dark/light mode
- Smooth animations and cursor glow effect
- Mobile-first, fully responsive

### 🔍 Interactive Features
- Course search with real-time filtering
- URL-based course routing (`?course=react`)
- Infinite scrolling tech strip
- Scroll snap sections

### 📊 Data-Driven Architecture
- All course content in a single JavaScript data file
- Easy to add new courses
- No database or backend required

---

## 🛠️ Tech Stack

| Technology | Role |
|------------|------|
| HTML5 | Semantic markup |
| CSS3 | Styling, animations, responsive design |
| JavaScript (ES6+) | Interactivity, data management |

**Zero Dependencies** — No frameworks, no libraries, no build tools!
